Frases HTML

EM: Se utiliza para dar significado a una parte del texto.

STRONG: se utiliza para resaltar

CODE: se usa para poner el texto como si fuera código de algun lenguaje de programación

Q: Se utiliza para poner Comillas

ABBR: Sirve para poner abreviaciones

ACRONYM: Sirve para poner acrónimos

KBD: Texto que debe ser introducido por el usuario

SAMP: Se usa para mostrar un ejemplo de una salida de un programa

VAR: se utiliza par amostrar el texto de una variable

DFN: sirve para poner definiciones
